# 📧 Email Spam Detection using Machine Learning

This project is a Machine Learning based Email Spam Detection system that classifies messages as Spam or Not Spam using Natural Language Processing techniques and a trained ML model.

The application is deployed using Streamlit, which allows users to enter a message and instantly get spam detection results.

---

## 🚀 Features

- Detects spam emails or SMS messages
- Uses Natural Language Processing (NLP)
- TF-IDF Vectorization
- Machine Learning Model Prediction
- Interactive Streamlit Web Interface
- Real-time message classification

---

## 🧠 Technologies Used

- Python
- Streamlit
- Scikit-learn
- NLTK
- Pickle
- TF-IDF Vectorizer
- Machine Learning Model
